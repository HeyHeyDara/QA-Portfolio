# BR-001 Information Pages Are Empty

## Title
Information pages do not contain any content

## Environment
- Browser: Google Chrome 138
- OS: Windows 11
- Website: https://demo.opencart.com

## Preconditions
User is on the homepage.

## Steps to Reproduce
1. Scroll down to the footer.
2. Open "Terms & Conditions".
3. Open "Delivery Information".
4. Open "About Us".
5. Open "Privacy Policy".

## Expected Result
Each information page should contain relevant information for users.

## Actual Result
All pages open successfully but contain only the page title without any informational content.

## Severity
Medium

## Priority
Medium
